# China Hat Module

A simple **China Hat cosmetic module** for **Meteor Client addons** that renders a conical hat above the player's head.

The hat smoothly follows the player's position and can be customized with colors.

---

# Features

- Smooth **3D China Hat rendering**
- Custom **top and bottom colors**
- **First person visibility toggle**
- Interpolated movement (no jitter)
- Lightweight rendering

---

# Settings

### Show In First Person
Allows the hat to render while using **first person view**.

-

### Bottom Color
Controls the **color of the hat's edge**.
